# CafeNoir

CafeNoir is a simple cafeteria management system written in C# as a group project for the [2022 EpsilonNet Coding School](https://res.epsilonnet.gr/marketing/CodingSchool/No2/index_No2.html). It was written in C#, using modern .NET and Windows Forms.
